# Portfolio Content

This context defines how Veerakran's professional work is described without overstating responsibility or exposing confidential information.

## Language

**Public case study**:
Professional work that can be supported with public project details, media, or links.
_Avoid_: Featured project, main project

**Professional project experience**:
Substantial professional work whose contribution and engineering context can be described, while client and organisation identities or sensitive project details remain undisclosed. It is equal in importance to a public case study, but has less public evidence available.
_Avoid_: Other work, NDA project, smaller engagement, secondary project

**Workstream**:
A grouped body of related professional responsibilities used when the available facts cannot safely or accurately be divided into individual client projects.
_Avoid_: Anonymous project, unnamed project
